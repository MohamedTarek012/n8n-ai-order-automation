# AI-Powered Order Processing Automation (n8n)

## 🚀 Overview
This project automates order processing and reporting using n8n, Shopify APIs, and AI Agent.

## ⚙️ Features
- Automated data extraction from Shopify (REST & GraphQL APIs)
- Data transformation and filtering using JavaScript
- Dynamic report generation (Chef / Butcher / Orders)
- Integration with Google Sheets & Google Drive
- AI Agent (LLM) generates operational instructions
- Scheduled workflows (every 2 days)

## 🧠 AI Component
An AI Agent analyzes structured data and generates actionable instructions for kitchen operations.

## 🛠 Tech Stack
- n8n
- Shopify API
- JavaScript
- Google Sheets API
- LLM (Gemini / OpenAI)

## 📂 How to Use
1. Import the JSON workflow into n8n
2. Configure credentials (Shopify, Google, Email)
3. Run the workflow
